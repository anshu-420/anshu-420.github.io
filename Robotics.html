Using the Raspberry Pi


In 10th grade I was I took a course that my school offered, and it was called Computer Engineering.
It introduced to the world of electronics, and how we are able use code to manipulate electronics to do
various tasks. This was the first time I was able to use the Raspberry Pi 3, a small computer, thatis
programmed to teach computing and use electronics create projects. My favourite project were using the
concepts of robotics to build a autonomous buggy car. From that day forth, I've tried to build different types
of robotics outside of class and join contest with friends to challenge ourselfves.
